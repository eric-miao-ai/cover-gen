# Scene Illustration Example

```text
Use $cover-gen 生成一张卡通图：图书馆里一个女孩正在看电脑查资料，氛围专注安静。
```

## Intended Extraction

- use_case: scene-illustration
- scene: library
- character: girl
- action: reading and researching on a computer
- mood: focused, quiet

## Expected Output Shape

A calm cartoon scene that preserves the explicit library setting while keeping the same clean house style and simplified modern environment treatment.
